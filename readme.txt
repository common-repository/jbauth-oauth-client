JBAuth OAuth allows users to login/register into your wordpress using their JBAuth account, with shortcodes support.

== Description ==

JBAuth OAuth allows users to login or optional register into your wordpress site using their JBAuth account, it will place a login button on your login page and uses shortcodes to put the button wherever you want it to show, even menus

Major features in JBAuth OAuth include:

* Add a button to the login page of your installation.
* Allow to locate the login button wherever user wants with shortcodes.
* Allow to set if user can register with their JBAuth credentials if the user doesn't exist on your installation or just an error login message.

== Installation ==

Upload the JBAuth OAuth plugin to your installation, Activate it, then enter your client id, Secret and API Key retrieved from the JBDevelopers Dashboard https://dev.justinback.com/.

== Frequently Asked Questions ==
Q: Where can I create an app?
A: https://dev.justinback.com/dashboard/oauth_new_app.php

Q: Where can I create an apikey?
A: https://dev.justinback.com/dashboard/new_apikey.php

Q: Where can I get a verified app?
A: https://wiki.justinback.com/Client_Verification

Q: My API Key has been suspended!
A: Contact the JBDevelopers support team by sending an email to support@justinback.com

Q: My App is untrusted!
A: This is due to a change from a verified application. Contact the JBDevelopers support team by sending an email to support@justinback.com
